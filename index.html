Simeji is IME to help user communicate more uniquely

  • Role Designer
  • Date2013-2016
  • Sitesimeji.me/
  • AwardApple Best of 2014 Japan
Simeji is IME that provides unique communciation ways for the users. For example, it provides cloud base text prediction which dictionary updated over the time based on the user trends. It was initially developped for Android which had over 14 million users by 2014. And in 2014, Responding to that iOS 8 accepts the 3rd party IME for iOS, we ddeveloped the app for iOS in 3 months to release. At the following year, we updated its android design to adapt material design.

To release iOS version in 3 months in 2014, we selected the feature carefully, considering its strength in the market.
