Type with Processing

Processing is open source software with which coders can interact with its visual outputs. All types in this project are generated digitally with numbers, angles, or sign/cosign calculation etc. It's on GitHub with MIT License.

Ways to say

This is a chatbot that works on Facebook messenger to give feedback to what the user is about to say. The background of this idea is increasing diversity around. I notice sometimes people say something offensive to another without actually knowing it is not a nice way to say. This app is for such users to be aware and suggests alternative ways to say. The development process is shared on Github.

Rest... prototype with Framer

This is a prototype (iOS 6Plus) of a function that suggests users to rest and reflect own psychological status. An idea behind this prototype is that something to help people rest and reflect a day maybe useful for healthy life. This idea was derived mainly from two previous studies: 'The best interface is No interface’ by Golden Krishna and an app 'Moodnotes' developed by ThrivePort, LLC. Particularly, an element about dark interface to calm down is from the book and recording self psychological pattern is from the app.

Natural light and software

“Light is the most important synchronizing agent for the brain and body,”
said Ivy Cheung, co-lead author and Ph.D. candidate in neuroscience Northwestern University. (Cited from here) Such studies let me think possibility in combining healthcare software and morning light, bright blue sky light, cloudy day light, or evening darkness. While thinking this light/health issue, f.lux became an inspiration to mockup this conceptual work.

Exploring with Google Maps

Which information is indispensable? It is the question asked throughout this little project exploring with Google Maps. Imaging the time people use handwritten map on paper to show recommendation to visitors, I gave hierarchy among information appeared on the map. It's on GitHub with MIT License.

Project management tool

Project management tool sample that can share the project issues and related tasks, based on based on Django. I think many project management have got complicated and we are not necessarily need all the functions provided. Rather, I think it is important to think what we need to communicate with team, remembering that human can process certain number of information at the same time. GitHub with MIT Licence.
